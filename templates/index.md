# Templates

Шаблоны используются для повторяемого производства материалов из claims, sources и evidence.

## Основной принцип

Шаблон не разрешает придумывать факты. Если данных нет, в материале остается `TODO: нужны данные RikaNV`.

## Список шаблонов

- product-profile;
- claim;
- source;
- evidence;
- test-report;
- comparison-article;
- buyer-guide;
- faq-page;
- service-article;
- firmware-release-note;
- youtube-script;
- telegram-post;
- vk-dzen-post;
- dealer-material;
- council-review;
- ai-answer-evaluation;
- content-atom.
