# Anti-Spam Checklist

- [ ] Нет рерайт-мусора.
- [ ] Нет повторения одной статьи на разных сайтах без новой функции.
- [ ] У каждой публикации есть уникальная задача.
- [ ] Не создан фейковый независимый ресурс.
- [ ] Нет PBN.
- [ ] Нет дорвеев.
- [ ] Нет покупных обзоров без маркировки.
- [ ] Нет фейковых отзывов.
- [ ] Нет keyword stuffing.
- [ ] Канонический источник указан.
- [ ] Материал помогает пользователю, а не только закрывает запрос.
