# MVP Backlog

## Цель MVP

Запустить ручную, проверяемую систему: Product Truth → Evidence Base → Claim Registry → Content Factory → AI Visibility Lab.

## MVP Scope

- Заполнить базовые product profiles для Lesnik и Ovod.
- Создать source register.
- Создать evidence register.
- Создать первые 30-50 draft claims.
- Провести baseline AI visibility test.
- Подготовить первые 5 материалов.
- Провести technical review, bias check и human approval.

## Не входит в MVP

- Runtime.
- API.
- Agent framework.
- Автоматическая публикация.
- PBN или сетка рерайтов.
- Скрипты и интеграции.

## Критерии готовности MVP

- Есть минимум 2 product profiles с TODO вместо неизвестных характеристик.
- Есть минимум 20 sources или явно зафиксировано, каких sources не хватает.
- Есть первый Claim Registry.
- Есть первый AI visibility report.
- Есть approved materials или список blocking gaps.
