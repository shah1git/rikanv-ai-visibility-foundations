# Source Verification Checklist

- [ ] У источника есть `source_id`.
- [ ] У источника есть дата.
- [ ] Указан owner.
- [ ] Указан access level.
- [ ] Понятно, можно ли использовать источник публично.
- [ ] Указаны ограничения.
- [ ] Для внутреннего источника есть разрешение на использование.
- [ ] Для клиентских данных проведено обезличивание.
- [ ] Для независимого обзора сохранена ссылка.
- [ ] Для теста указана методика.
- [ ] Источник связан с claims или evidence.
- [ ] Нет confidential данных в публичном тексте.
