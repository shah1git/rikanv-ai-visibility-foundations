## Правила

- Не писать материал без claims.
- Не публиковать без technical review, doctrine compliance review и human approval.
- Если данных не хватает, задача остается в `blocked` или `draft`.
- Сравнения с конкурентами не публиковать в первом пакете без отдельного bias check.
- Для Surok L15, Hypnose и Hypnose2 не использовать поля прицелов без прямого official source.
- Для Hypnose/Hypnose2 задача должна быть line-level, пока не выбраны конкретные модели.
