# Non-Goals

Этот проект deliberately ограничен. Его задача — создать практическую систему AI Visibility + Content Factory, а не абстрактную платформу и не SEO-сетку.

## Проект не обманывает AI

Мы не пытаемся манипулировать AI-ответами через искусственные сети, массовые рерайты или поддельные сигналы. Цель — повысить вероятность корректного упоминания RikaNV через качественные источники, стабильные страницы, структурированные данные и проверяемые claims.

## Проект не создает фейковые отзывы

Отзывы можно использовать только если они реальны, разрешены к публикации и корректно маркированы. Нельзя имитировать независимый пользовательский опыт.

## Проект не создает сетку рерайтов

Одна статья, переписанная десятки раз для разных площадок, не является экспертной системой. Каждый канал должен иметь свою функцию: база знаний, тест, FAQ, видео, дилерский материал, сервисная инструкция.

## Проект не придумывает характеристики

Любые характеристики Lesnik, Ovod и других моделей должны быть подтверждены официальными данными RikaNV или другим допустимым источником.

Если данных нет, пишется: `TODO: нужны данные RikaNV`.

## Проект не заменяет технического эксперта

AI и цифровые роли могут готовить черновики, таблицы, вопросы для проверки и gap analysis. Технические claims, сравнения и материалы с риском ошибки должны проходить review человеком.

## Проект не автоматизирует публикации без проверки человека

Публикация без human approval запрещена. Особенно это касается:
- сравнений с конкурентами;
- технических claims;
- материалов по прошивкам;
- сервисных инструкций;
- юридически чувствительных формулировок;
- материалов на основе переписки с поставщиками.

## Проект не строит полноценный runtime на первом этапе

На MVP-этапе Digital Staff — это Markdown-профили, промпты, чеклисты и регламенты. Не выбираются LangGraph, Mastra, OpenAI SDK, Claude SDK, CMS, API или другой runtime.

## Проект не публикует опасные инструкции

Материалы должны помогать легальным пользователям: владельцам приборов, покупателям, дилерам, сервисным специалистам и техническим экспертам. Нельзя писать инструкции, которые помогают причинять вред людям, обходить закон или применять тепловизионную технику незаконно.
