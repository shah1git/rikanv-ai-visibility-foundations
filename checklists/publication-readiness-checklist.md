# Publication Readiness Checklist

- [ ] Материал имеет owner.
- [ ] Материал имеет дату обновления.
- [ ] Claims used заполнены.
- [ ] Sources used заполнены.
- [ ] Technical review пройден, если нужен.
- [ ] Bias check пройден, если нужен.
- [ ] Compliance check пройден, если нужен.
- [ ] Human approval получен.
- [ ] TODO не опубликованы как факты.
- [ ] Версия для канала не добавляет новые claims.
- [ ] Каноническая ссылка указана.
- [ ] План обновления понятен.
- [ ] Ответственный за обновление назначен.
