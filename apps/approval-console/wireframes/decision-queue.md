# Wireframe: Decision Queue

## Purpose

Let reviewers make structured decisions without reading all Markdown files.

## Layout

```text
+----------------------------------------------------------+
| Decision Queue                                           |
+----------------------------------------------------------+
| HRD-001  Ovod L25 no-LRF wording            P1 high     |
| Issue: limitation wording needs neutral handling         |
| Recommendation: factual source-backed wording            |
| Options: [A] [B] [C selected] [D]                       |
+----------------------------------------------------------+
| HRD-004  Detection/LRF wording              P0 blocker  |
| Recommendation: official-spec wording only               |
| Options: [A selected] [B] [C]                           |
+----------------------------------------------------------+
```

## Decision Card Fields

- topic;
- priority;
- related materials;
- issue;
- agent recommendation;
- options;
- default safe action;
- risk if approved;
- risk if rejected.
