# Digital Staff

Digital Staff — операционная модель цифровых ролей для RikaNV AI Visibility + Content Factory.

После agent-first pivot Digital Staff является базовой моделью пилота: роли
выполняют импорт, извлечение, review, draft и repurposing. Это не runtime и не
agent framework.

## Роли

- Site Import Agent
- Product Truth Agent
- Claim Extraction Agent
- Evidence Linker Agent
- Doctrine Review Agent
- Technical Consistency Agent
- Content Draft Agent
- GEO/SEO Agent
- Publisher/Repurposer Agent
- AI Visibility Analyst Agent
- Content Strategist
- Researcher
- Writer
- Comparison Analyst
- Technical Reviewer
- GEO/SEO Specialist
- Publisher Repurposer
- AI Visibility Analyst
- Service Knowledge Assistant
- China Assistant

## Правило разделения обязанностей

Ни одна роль не может одновременно писать, технически проверять, утверждать и публиковать один и тот же материал.

Агент может готовить, но не может сам финально публиковать. Human publication
approval обязателен.

## MVP-реализация

На первом этапе каждая роль — это Markdown-профиль, промпт, чеклист и место в
workflow. Это не runtime и не agent framework.
