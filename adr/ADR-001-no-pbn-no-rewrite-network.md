# ADR-001: No PBN, No Rewrite Network

## Status

accepted

## Context

Для роста видимости можно было бы использовать PBN, дорвеи, массовые рерайты и поддельные независимые сайты. Эти подходы создают краткосрочные сигналы, но вредят доверию.

## Decision

RikaNV AI Visibility Foundations не строит PBN и сеть рерайтов.

## Rationale

- Риск для доверия пользователей.
- Риск для поисковых систем.
- Риск для бренда.
- Риск несогласованных и устаревших claims.
- Лучше строить экосистему полезных источников: официальный сайт, база знаний, тесты, FAQ, видео, дилерские материалы и честные внешние обзоры.

## Consequences

### Positive

- Устойчивый рост доверия.
- Меньше репутационных рисков.
- Контент можно проверять и обновлять.

### Tradeoffs

- Эффект может быть медленнее.
- Нужны реальные sources, evidence и review.
