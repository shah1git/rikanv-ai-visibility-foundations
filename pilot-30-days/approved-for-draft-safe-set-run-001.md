# Approved-for-Draft Safe Set Run 001

## Purpose

Этот файл содержит claims, которые агенты могут использовать в первых черновиках.

Они НЕ approved for publication.
Финальная публикация требует human approval.

## Inclusion rules

Claim включается, если:

- он был `auto_approved_for_draft`; или
- он был `needs_rewording`, но safe exception resolution разрешил neutral official-spec wording;
- имеет source_id;
- имеет evidence_id;
- имеет neutral wording;
- не требует unresolved human exception;
- не является volatile commercial claim;
- не является competitor comparison;
- не является field performance claim without test.

## Exclusion rules

Claim исключается, если:

- human exception unresolved;
- needs evidence;
- risky wording cannot be safely neutralized;
- line/model ambiguity unresolved;
- volatile commercial data;
- performance claim without field evidence.
