# Wireframe: Runs

## Purpose

Show all agent runs that require or recently received human attention.

## Layout

```text
+----------------------------------------------------------+
| RikaNV AI Console                         [Import JSON]  |
+----------------------------------------------------------+
| Runs                                                     |
|                                                          |
| [Run 001] Decision Applied    4 materials   7 decisions |
| [Run 002] Draft Cleanup       pending       3 blockers  |
| [Run 003] AI Retest           later         no action   |
+----------------------------------------------------------+
```

## Primary Actions

- Open run dashboard.
- Import approval data JSON.
- Export human decisions JSON after review.
