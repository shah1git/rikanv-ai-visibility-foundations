# Product Truth

Product Truth — канонический слой данных о продуктах RikaNV. Он должен быть заполнен до масштабного производства контента.

## Что хранится здесь

- Профили продуктов.
- Реестр claims.
- Реестр sources.
- Реестр evidence.
- Шаблоны карточек моделей.
- Ссылки на тесты, сервисные заметки, прошивки и дилерские материалы.

## Правила

- Не заполнять характеристики из памяти.
- Все отсутствующие данные помечать как `TODO: нужны данные RikaNV`.
- Любой продуктовый claim должен иметь source и evidence.
- Product profile обновляется после новых тестов, прошивок, цен и изменений наличия.

## Начало работы

1. Скопировать `product-profile-template.md` для каждой модели.
2. Заполнить только подтвержденные поля.
3. Создать draft claims.
4. Перед публикацией перевести claims в `verified` через technical review.
