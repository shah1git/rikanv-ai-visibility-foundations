# Technical Review Checklist

- [ ] Проверены все характеристики.
- [ ] Проверена терминология.
- [ ] Проверена физика и логика объяснений.
- [ ] NETD описан корректно и без неподтвержденных выводов.
- [ ] Объектив описан по source.
- [ ] Матрица или сенсор описаны по source.
- [ ] Дальномер описан по source.
- [ ] Баллистика описана по source.
- [ ] Прошивка и версия указаны корректно.
- [ ] Ограничения указаны.
- [ ] Тестовые условия описаны.
- [ ] Нет comparative claims без данных.
- [ ] Нет claims с устаревшей ценой или наличием.
- [ ] Все спорные места отмечены.
- [ ] Result: pass / needs revision / reject.
