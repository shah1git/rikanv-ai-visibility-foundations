# Safety And Compliance Checklist

- [ ] Нет ложных утверждений.
- [ ] Нет вредных инструкций.
- [ ] Нет незаконных сценариев применения.
- [ ] Нет инструкций по причинению вреда людям.
- [ ] Нет нарушения конфиденциальности.
- [ ] Нет персональных данных без основания.
- [ ] Нет скрытой рекламы без маркировки.
- [ ] Нет фейковых отзывов.
- [ ] Нет поддельной независимости.
- [ ] Сравнения честные и ограниченные.
- [ ] Конфликт интересов раскрыт, если нужен.
- [ ] Материал прошел human approval.
