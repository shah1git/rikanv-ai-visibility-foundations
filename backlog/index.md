# Backlog

Backlog хранит первые задачи для запуска RikaNV AI Visibility + Content Factory.

## Разделы

- MVP.
- Первые 30 статей.
- Первые 20 сравнений.
- Первые 10 тестов.
- Первые 50 AI-запросов.
- Данные по продуктам.
- Сервисные знания.
- Дилерские материалы.

## Правила

- Backlog не является публикационным планом без review.
- Любая тема должна быть связана с claims и evidence.
- Сравнения не содержат финальных выводов без тестов.
