# Comparison Bias Checklist

- [ ] Материал не выглядит как реклама.
- [ ] Указаны сильные стороны RikaNV только при наличии evidence.
- [ ] Указаны слабые места RikaNV или missing data.
- [ ] Указаны сильные стороны конкурента.
- [ ] Не скрыты важные параметры.
- [ ] Сравниваются модели одного класса или отличие объяснено.
- [ ] Нет абсолютного вывода без данных.
- [ ] Таблица не манипулирует порядком или выбором параметров.
- [ ] Источники конкурента указаны.
- [ ] Вывод сделан по сценариям.
- [ ] Есть блок “каких данных нет”.
- [ ] Есть technical review.
- [ ] Есть human approval перед публикацией.
